# Project Title
 
Mini Project 2 Obesity Graph
 
## Description
 
The program looks though a patient data set and plots a graph on the number of healthy vs obese patients
 
## Getting Started
 
### Dependencies

Download the dataset from https://www.kaggle.com/datasets/prasad22/healthcare-dataset?resource=download

Then create a data folder within your main branch and extract the csv file into it

Please install the pip requirements:
```
pip install -r requirements.txt
```
 
### Executing program
 
```
python main.py
```
